# PERSONAL — Личная AI-продуктивность

## Скиллы

### [jtbd](skills/jtbd.md) — Анализ интервью по методологии AJTBD
**Когда триггерится:** есть транскрипт интервью, нужно понять джобы аудитории, выбрать сегмент или разобрать switch  
**Что делает:** извлекает джоб по формуле "когда / хочу / чтобы", строит граф работ, проводит switch-анализ (раздражение → поиск → решение), определяет механику создания ценности  
**Контекст:** транскрипт интервью + тип (найм / switch / отток) + вопрос для ответа

## Задачи, под которые планирую найти / собрать скиллы

| Задача | Что нужно |
|---|---|
| Быстро описать продукт для нового партнёра | Скилл: product pitch по шаблону (проблема / решение / ЦА / механика / рвы) |
| Провести конкурентный анализ | Скилл: сравнение по слоям (ЦА / джобы / механики ценности / рвы) |
| Приоритизировать фичи | Скилл: RICE или impact/effort с учётом джобов текущего сегмента |
| Написать контент для B2B-партнёра | Скилл: адаптация продуктового нарратива под боли B2B (не фичи, а ROI и switching cost) |
| Разобрать негативный фидбек | Скилл: классификация жалоб по типу (продукт не выполняет работу / ожидания не совпали / привычка сильнее) |
