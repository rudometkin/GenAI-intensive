# PRODUCT.md — FOOD2MOOD

## О продукте

**FOOD2MOOD** — AI-ассистент для онлайн-меню ресторанов. Встраивается в QR-меню как плагин и помогает гостю выбрать блюда под его текущее настроение, пищевые ограничения и уровень голода.

**Модель монетизации:** B2B — продаём плагин сервисам QR-меню, которые перепродают его своим ресторанам или включают как конкурентное преимущество в тариф.

---

## Целевая аудитория

### Конечные пользователи (B2C — гости ресторанов)

| Сегмент | Описание | Ключевая боль |
|---|---|---|
| Аллергики и люди с ограничениями | Непереносимости лактозы, глютена, религиозные ограничения | Страх ошибиться в выборе, долгое изучение состава |
| Следящие за питанием | Считают КБЖУ, на диете, следят за белком | Нет быстрого способа фильтровать по нутриентам |
| Гости в незнакомом ресторане | Впервые в заведении или в непривычной кухне | Не понимают состав и вкус блюд по названию |
| Нерешительные | Информационная перегрузка при большом меню | Хочется готового совета "под меня" |

### Прямые клиенты (B2B — партнёры)

**Сервисы QR-меню** — платформы, через которые рестораны создают и управляют своим цифровым меню. Примеры: Poster, iiko, Syrve, локальные решения.

Им важно: дифференциация от конкурентов, рост конверсии в заказ у ресторанов-клиентов.

---

## AI-агрегаторы, которые используем

| Инструмент | Зачем |
|---|---|
| **Claude API (Anthropic)** | Планируется для v2.0 — генерация персонализированных описаний блюд и свободный диалог с гостем |
| **Текущая v1.0** | Rule-based алгоритм без LLM — keyword matching + атрибуты блюд |

В текущей версии прототипа LLM не используется намеренно: нужно было показать механику UX без зависимости от внешнего API и задержек.

---

## Парсеры, которые используем

На текущем этапе (демо-прототип) внешние парсеры не используются — меню подаётся партнёром через конфигурацию плагина.

**В планах:**
- Парсинг меню с сайтов ресторанов для онбординга новых партнёров без ручного ввода
- Парсинг отзывов (Google Maps, 2GIS) для наполнения базы болей аудитории и улучшения тегов рекомендаций
