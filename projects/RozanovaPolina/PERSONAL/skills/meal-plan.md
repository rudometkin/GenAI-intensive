# meal-plan

**Триггер:** «спланируй питание на неделю» / «что готовить» / «составь меню» / «батч-готовка»

## Что делает

1. Читает принципы нутрициологии из базы знаний
2. Читает доступные рецепты и заготовки
3. Строит батч-план готовки на неделю (воскресенье = день готовки)
4. Распределяет по контейнерам с маркировкой (1А, 2Б…)
5. Формирует список покупок

## Контекст

```
Читать перед планированием:
- Нутрициолог/Читаемая база/00_Главный_свод_принципов.md
- Нутрициолог/Читаемая база/01_Как_строить_меню.md
- Нутрициолог/Читаемая база/02_Метод_тарелки_и_баланс.md
- Кулинария/Принципы заготовок - читаемая база/ (нужные файлы)
- Кулинария/Рецепты - читаемая база/kbzhuy_recipes_structured.md
```

## Принципы (из базы знаний)

- Метод тарелки: 50% овощи, 25% белок, 25% сложные углеводы
- Батч = один день готовки → 5–7 дней без усилий
- Контейнеры нумеруются: цифра (день) + буква (приём пищи: А=завтрак, Б=обед, В=ужин)
