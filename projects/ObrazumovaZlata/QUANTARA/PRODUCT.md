# PRODUCT — Квантара

## Агрегаторы нейросетей

На текущем этапе внешние AI-агрегаторы не используются. В плане — RAG-слой поверх Supabase pgvector для поиска по протоколам экспериментов.

## Парсеры

Внешние парсеры не используются. Данные вводятся через интерфейс или загружаются через Supabase Storage.

## Продукт и целевая аудитория

**Что такое Квантара:**
ELN + LIMS (Electronic Lab Notebook + Laboratory Information Management System) для российских научных команд. Единая платформа для управления проектами, задачами, реагентами, расходниками и протоколами экспериментов.

**Позиционирование:**
Первое отечественное полнофункциональное ELN+LIMS-решение. Зарубежные лидеры (Benchling, LabWare, STARLIMS) недоступны в РФ с сентября 2024 г. из-за санкций. Российские аналоги (1С:LIMS, myLab) покрывают только LIMS без ELN и без AI.

**Сегменты (SOM ₽100–150 млн ARR за 3–5 лет):**

| Сегмент | Целевой ARR |
|---|---|
| Фарма и биотех (GxP compliance) | ₽40–60 млн |
| НИИ и университеты (~800 адресуемых) | ₽17–24 млн |
| ЦКП (центры коллективного пользования, ~500) | ₽12–20 млн |
| Промышленный R&D (нефтехим, матвед) | ₽10–15 млн |
| Hardware / IT R&D (микроэлектроника) | ₽10–14 млн |

**Боли:**
- Учёт реагентов и расходников в Excel → потери, просроченные запасы
- Нет трекинга задач и дедлайнов внутри лаборатории
- Нет истории изменений (критично для GxP-аудита)
- Зарубежные аналоги заблокированы после санкций 2024 г.
- 200 новых молодёжных лабораторий запускаются с нуля — нет legacy, готовы к новому ПО

**Рыночный контекст:**
- TAM глобальный: ₽320 млрд ($3,5 млрд, CAGR 8–12%)
- SAM Россия: ₽1–3 млрд, ~3 200 адресуемых организаций, рост >15%/год
- Катализаторы: 0 иностранных конкурентов, госсубсидии до 50% стоимости внедрения (₽6 млрд выделено), рост НИОКР +14% в 2024 г.
