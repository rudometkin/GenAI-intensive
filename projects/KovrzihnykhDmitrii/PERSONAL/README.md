## План: какие ещё скиллы собрать

Набор задач, под которые планирую собрать/найти персональные скиллы:

1. **`menu-parser`** — тащить меню ресторана из разных форматов (PDF, Excel, фотка, веб-страница) в единую схему `dish → ingredient`. Пока делаю вручную в `app/pipelines/txt_data_layer.py`, хочется отдельно и переиспользуемо.
2. **`ajtbd-extractor`** — из 30-минутного интервью с пользователем собирать карту `job → situation → outcome → anxiety → motivation`. Сейчас делаю в заметках, нужна воспроизводимая рубрика.
3. **`moat-audit`** — по произвольному сервису/моему проекту возвращать состояние рвов по 5 осям (данные / доверие / сеть / switching cost / регуляторка) и ближайший следующий шаг. Методологический — как чеклист для ретро.
4. **`interview-script`** — генерирует сценарий problem-интервью под конкретную гипотезу (без leading questions). Нужен для поиска ЦА Food2Mood среди сетевых ресторанов.
5. **`growth-hook`** — пишет 10 разных вариантов хука для соцсетей под одну и ту же новость / фичу (vniklay + Food2Mood). Сейчас пишу вручную, это узкое горлышко.
6. **`srs-word`** — формирует карточку Anki/SRS по фразе, выделенной из клипа: контекст, перевод, grammar tag, мнемоника. Нужен, когда vniklay дойдёт до модуля повторений.
7. **`adr-writer`** — развитие `fiksiruy`: по разговору собирает оформленный ADR-файл, а не просто запись в логе.
8. **`dataset-smoke`** — быстрый smoke-test произвольного датасета: распределения, выбросы, дубли, утечки — на тот же вход, что и модель.
9. **`cheap-ml-infra`** — оценивает, где дешевле всего гонять следующий эксперимент (локально / RunPod / Vast.ai / Yandex DataSphere), с учётом данных, которые уезжают за периметр.
10. **`competitor-watcher`** — раз в неделю собирает изменения у 10 отобранных конкурентов (релизы, посты, вакансии) и даёт дельту.
