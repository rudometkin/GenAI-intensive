# PRODUCT — О продукте Roomy

## Агрегаторы нейросетей

| Агрегатор | Зачем используем |
|-----------|-----------------|
| **Replicate** | Запуск Stable Diffusion img2img для генерации интерьеров — быстрый старт без своей GPU-инфраструктуры |
| **Claude API (Anthropic)** | Парсинг и нормализация данных о мебели, анализ фото комнаты (размеры, стиль, состав), генерация описаний |
| **OpenAI GPT-4o** | Vision-анализ загружаемых фото: определение стиля комнаты, распознавание мебели на фото |

## Парсеры

| Парсер / Инструмент | Зачем |
|--------------------|-------|
| **Playwright** | Автоматический сбор каталога с сайтов Hoff, IKEA — SKU, размеры, цены, фото товаров |
| **Firecrawl** | Структурированный парсинг страниц товаров с нормализацией через LLM |
| **Bright Data** | Ротация прокси для обхода блокировок при парсинге крупных ретейлеров |

## Продукт и целевая аудитория

### Что это за продукт

**Roomy** — AI-сервис дизайна интерьера с реальной мебелью. Пользователь загружает фото комнаты, указывает её размеры и бюджет → Roomy подбирает мебель из реального каталога с учётом габаритов и генерирует визуализацию "как будет выглядеть" с конкретными товарами.

Ключевое отличие от конкурентов (InteriorAI, Reimagine Home): мы работаем с реальными размерами и реальными товарами, которые можно купить прямо из приложения. Конкуренты генерируют красивые картинки с несуществующей мебелью.

### Целевая аудитория

**Сегмент 1 — Переезжающие (основной)**
- 25–40 лет, крупные города (Москва, СПб, Екатеринбург)
- Новая квартира / ремонт, бюджет 200–800k на обстановку
- Боль: боятся купить не то, тратят время на замеры и IKEA-планировщик
- Канал: таргет в Instagram/VK, SEO по "дизайн комнаты онлайн"

**Сегмент 2 — Обновляющие интерьер**
- 28–45 лет, уже обустроенная квартира
- Хотят обновить 1–3 предмета, не знают как вписать в существующий интерьер
- Боль: боятся что новая вещь не впишется, не могут визуализировать
- Канал: органика, рекомендации

**Сегмент 3 — Экономные на дизайнере**
- 30–55 лет, есть бюджет на мебель, нет на дизайнера
- Хотят профессиональный результат без переплаты
- Боль: дизайнер = дорого и долго
- Канал: SEO "дизайн интерьера онлайн бесплатно", контент-маркетинг
