# PLAN — Построение рвов на 3–6 месяцев

## Ров 1: Данные

**Что делаем:**
- Собираем структурированные данные о комнатах пользователей: фото + размеры (площадь, высота потолков, тип планировки)
- Собираем данные о предпочтениях: какие стили выбирают, какую мебель лайкают, что покупают
- Формируем датасет пар "комната → подходящие товары" для fine-tuning рекомендательной модели
- Через 3 месяца: 10 000+ аннотированных комнат → точность подбора растёт

**Метрика:**
- Размер датасета комнат (цель: 10k за 3 мес.)
- CTR на рекомендованные товары (цель: >15%)
- Конверсия в покупку через партнёрские ссылки (цель: >3%)

**Что мешает:**
- Пользователи могут не вводить размеры (UX барьер) → делаем автодетект из фото
- GDPR/152-ФЗ: нужно явное согласие на использование фото для обучения

---

## Ров 2: Switching cost (профиль комнаты)

**Что делаем:**
- Каждый пользователь создаёт "комнату" с параметрами: размеры, стиль, бюджет, текущая мебель
- Все генерации привязаны к комнате → история вариантов сохраняется
- Коллаборация: можно поделиться комнатой с партнёром/дизайнером
- Через 6 месяцев: пользователь с 5+ сохранёнными комнатами и историей генераций не уйдёт к конкуренту

**Метрика:**
- % пользователей с 2+ сохранёнными комнатами (цель: 40%)
- Retention D30 (цель: >25%)
- Количество "расшаренных комнат" (вирусность)

**Что мешает:**
- Пользователи не хотят создавать аккаунт → делаем "гостевой" режим с миграцией в аккаунт
- Первая комната создаётся с трением (нужно вводить данные) → упростить онбординг

## Ров 3: Сеть (партнёрская экосистема)

**Что делаем:**
- Интегрируем каталоги Hoff, IKEA, Askona, Lazurit: реальные SKU с ценами и 3D-моделями
- Партнёры платят CPA за переходы и покупки → Roomy монетизирует без платного доступа для пользователей
- Для пользователя: чем больше партнёров, тем больше вариантов → ценность растёт
- Для партнёра: чем больше пользователей, тем выше конверсия → партнёры хотят быть в каталоге
- Через 6 месяцев: двусторонняя сеть, где уход одного крупного партнёра — это переговорная позиция, а не катастрофа

**Метрика:**
- Количество партнёров в каталоге (цель: 5 за 3 мес., 15 за 6 мес.)
- GMV через партнёрские ссылки (цель: 1 млн руб./мес. за 6 мес.)
- NPS среди партнёров (цель: >50)

**Что мешает:**
- Крупные ретейлеры медленно подписывают договоры → начинаем с аффилиатных программ без договора (Hoff affiliate)
- Нужны 3D-модели товаров, у ретейлеров их нет → генерируем из фото товара
