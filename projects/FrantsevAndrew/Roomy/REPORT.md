# REPORT — Что изменилось за две недели

## Решения

- **Отказались от чистого img2img** как основного подхода — Stable Diffusion ломает геометрию комнаты и не понимает реальные габариты мебели. Решили разделить пайплайн: сначала 2D-планировщик (строгий геометрический расчёт), потом AI-рендер поверх планировки.

- **Выбрали Replicate вместо собственной GPU** для старта — экономит 3–4 месяца на инфраструктуру, при масштабировании переедем на RunPod или собственный кластер.

- **Парсинг через Playwright + Claude Vision** вместо классического HTML-парсера — структура сайтов ретейлеров меняется часто, LLM-нормализация устойчивее.

- **Hoff как первый партнёр**: начинаем с аффилиатной программы (без договора), это даёт возможность встроить реальный каталог уже сейчас.

## Сделано

- Базовый парсер каталога Hoff: 2 400 товаров с размерами, ценами, фото (категории: диваны, кресла, кровати, столы, стулья)
- MVP генерации: загрузка фото → выбор стиля → img2img через Replicate → результат
- Аутентификация через Supabase (Google OAuth)
- Хранение истории генераций в Supabase Storage
- Лендинг с формой для сбора ранних пользователей (47 заявок за неделю)

## Переделано

- **Убрали ручной ввод размеров**: пользователи не вводят — переходим на автодетект из фото через GPT-4o Vision. Точность ~70%, достаточно для MVP.
- **Переосмыслили онбординг**: убрали обязательную регистрацию, первая генерация — гостевая, регистрация — после просмотра результата.

- Расширение каталога: IKEA, Askona, Lazurit
