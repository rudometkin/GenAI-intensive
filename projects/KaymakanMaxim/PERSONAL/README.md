# PERSONAL — Личная AI-продуктивность

## Мои скиллы

### `social-content` — контент-план и посты под соцсети
**Триггер:** нужно создать посты, контент-календарь, адаптировать текст под платформу  
**Что делает:** генерирует платформо-специфичный контент (LinkedIn, Instagram, Telegram, ВК), строит контент-план под цели, адаптирует тон под аудиторию  
**Контекст:** читает `.agents/product-marketing-context.md` если есть — тогда вопросов не задаёт, сразу пишет  
**Как использую:** даю тему + платформу → получаю 5–10 вариантов постов с хуком, телом и CTA. Для контент-плана — цели + ЦА → 30 идей на месяц с рубриками

### `изобразика` — генерация изображений для постов (kie.ai)
**Триггер:** нужен визуал для поста, баннер, иллюстрация  
**Что делает:** генерирует изображения через kie.ai API (модели: Flux, Nano Banana, Seedream, Ideogram, GPT-4o image, Z-Image, Grok). Сохраняет в указанную папку  
**Как использую:** "создай обложку для поста про X в стиле Y" → получаю 3–5 вариантов → лучший в пост. Работает в паре с `social-content`: сначала текст, потом визуал под него

### `telegram-telethon` — парсинг и мониторинг Telegram
**Триггер:** нужно спарсить канал, мониторить ключевые слова, собрать данные из ТГ  
**Что делает:** работает через MTProto (Telethon) с user-аккаунта, умеет: история канала, live-мониторинг, участники, пересылка. Antiban по умолчанию (flood-wait, задержки, реалистичный девайс)  
**Как использую:** мониторинг конкурентов в нише SMM → сбор трендовых тем → в контент-план

## SMM-агент — задачи (проект в разработке)

Строим агента, который закрывает полный цикл SMM для малого бизнеса и агентств.

### Что агент должен уметь

| Задача | Статус | Инструмент |
|--------|--------|------------|
| Генерация контент-плана (30 постов/мес по теме + ЦА) | 🔄 в работе | Claude API + social-content |
| Написание текстов постов в голосе бренда | 🔄 в работе | Claude API |
| Генерация визуала под пост | ✅ есть | kie.ai / fal.ai |
| Автопостинг в Telegram | 🔜 следующий шаг | telegram-telethon |
| Парсинг конкурентов для идей | 🔜 следующий шаг | telegram-telethon |
| Сборка аналитики в готовый отчёт | ⬜ бэклог | LiveDune API + TG-STAT API |
| Интерпретация размытых правок клиента | ⬜ бэклог | Claude API |

### Архитектура (упрощённо)

```
Вход (тема / ЦА / бренд-голос)
      ↓
  Контент-план (30 идей)
      ↓
  Для каждого поста:
    ├── Текст (Claude API)
    └── Визуал (kie.ai / fal.ai)
      ↓
  Очередь публикации
      ↓
  Постинг (Telegram / ВК через API)
      ↓
  Аналитика → отчёт
```

### Боли, которые закрываем
- Агентство тратит 3–4 часа на сборку отчёта из 4+ источников → автоматический отчёт
- Клиент пишет "текст невкусный" → AI интерпретирует и применяет правки
- Малый бизнес не может контролировать SMM-специалиста → агент сам держит качество
