# RudometkinEgor

Пример структуры папки участника. Скопируйте её, переименуйте под себя (`SURNAME_NAME/`) и заполните файлы.

## Что внутри

- [PROJECTCLONE/](PROJECTCLONE/) — клонирование внешнего сервиса (Блок 1)
- [PERSONAL/](PERSONAL/) — личная AI-продуктивность: скилы + планы (Блок 2)
- [PROJECTNAME/](PROJECTNAME/) — ваш проект: код, отчёт, продукт, план рвов, AJTBD (Блок 3)
