# Skill: academic-writer

## Trigger
Академические тексты: рефераты, эссе, синопсисы (RU/EN)

## Function
- Структурирует текст: введение → основная часть → заключение
- Приводит стиль к академическому
- Проверяет логику аргументации

## Context
- Тип документа
- Объём
- Язык
- Дисциплина
- Ключевые тезисы

## Output format
- Формальный научный текст
- Чёткая структура
- Логические связки между абзацами
