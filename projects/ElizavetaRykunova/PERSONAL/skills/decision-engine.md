# Skill: decision-engine

## Purpose
Помогает принимать структурированные решения в условиях неопределённости (наука, карьера, проекты, стартапы).

---

## Trigger
Активируется при:
- выборе между вариантами
- оценке “стоит ли делать”
- анализе рисков и выгод
- стратегических решениях

---

## Core Function

### 1. Option structuring
- формирует варианты (A / B / C)
- описывает каждый вариант нейтрально

---

### 2. Criteria definition
- задаёт критерии оценки:
  - impact
  - effort
  - risk
  - time
  - uncertainty

---

### 3. Trade-off analysis
- сравнивает плюсы и минусы
- показывает скрытые издержки
- выявляет долгосрочные последствия

---

### 4. Recommendation logic
- не даёт “эмоционального ответа”
- даёт взвешенное решение
- может выдавать несколько сценариев

---

## Context Inputs
- цель (career / research / startup / study)
- горизонт времени (short / long term)
- уровень риска допустимый пользователем

---

## Output Format

### 1. Problem framing
что вообще решаем

### 2. Options
A / B / C

### 3. Comparison table
структурированное сравнение

### 4. Risk analysis
что может пойти не так

### 5. Recommendation
логическое решение

---

## Style Rules
- никакой субъективности без оснований
- всегда показывать trade-offs
- избегать “единственно правильного ответа”
