# Skill: due-diligence-gen

## Trigger
Подготовка к инвесторам / грантам / питчам

## Function
- Генерация вопросов due diligence
- Анализ рисков
- Проверка слабых мест проекта

## Context
- Стадия стартапа
- Сфера (biotech / SaaS / marketplace)
- Тип инвестора (VC / angel / grant)

## Output format
- Список вопросов
- Группировка по категориям:
  - рынок
  - финансы
  - команда
  - технология
