# PERSONAL — личная AI-продуктивность

## Мои скилы

### academic-writer
**Когда триггерится:** пишу реферат, эссе, синопсис, академический текст на русском или английском  
**Что делает:** структурирует текст по академическим требованиям (введение → основная часть → заключение), подбирает нейтральный научный стиль, проверяет логику аргументации  
**Контекст подкладывает:** тип документа, требуемый объём, язык, дисциплину, ключевые тезисы которые нужно раскрыть

---

### stats-helper
**Когда триггерится:** задача по статистике, t-тест, интерпретация p-value, работа с данными  
**Что делает:** объясняет какой тест применить и почему, пишет код на R/Python, форматирует вывод под требования сдачи (например, точное количество знаков после запятой)  
**Контекст подкладывает:** тип данных (одновыборочные / двухвыборочные), уровень значимости, формат вывода системы проверки

---

### pitch-builder
**Когда триггерится:** нужно написать питч, скрипт для презентации, investor deck  
**Что делает:** структурирует по формату problem → solution → market → traction → ask, адаптирует язык под аудиторию (инвесторы / жюри / клиенты)  
**Контекст подкладывает:** аудитория, формат (письменный / устный), ограничение по времени, стадия проекта

---

### vocab-translator
**Когда триггерится:** нужно перевести и объяснить научную терминологию (биология, экология, генетика)  
**Что делает:** переводит термины с контекстом, даёт примеры употребления в академических текстах, указывает на типичные ошибки перевода  
**Контекст подкладывает:** область науки, язык оригинала, язык перевода, уровень сложности

---

### due-diligence-gen
**Когда триггерится:** готовлю материалы для инвесторов, нужно предугадать вопросы  
**Что делает:** генерирует список вопросов due diligence по финансовой модели, рыночным рискам, команде, технологии  
**Контекст подкладывает:** стадия стартапа, сфера (biotech / SaaS / marketplace), тип инвестора (VC / angel / grant)
