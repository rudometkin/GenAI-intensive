# CODEBASE — Private Education

Продукт не предполагает публичной кодовой базы.

Ключевая ценность сервиса — **индивидуальный подход и личное присутствие**: работа ведётся один на один с семьёй, методология строится вокруг конкретного ребёнка, а не вокруг платформы. Автоматизировать это в виде продукта, доступного в интернете, — значит потерять главное конкурентное преимущество.
