# Skills

Набор инструментов, которые используются в повседневной работе над проектами.

---

## `presentation` — создание презентаций

Скилл для подготовки структурированных презентаций: питчи для инвесторов, учебные материалы, отчёты о прогрессе. Помогает выстроить логику повествования, сформулировать тезисы для каждого слайда и адаптировать подачу под конкретную аудиторию. Поддерживает несколько визуальных стилей — от минимализма до необрутализма. На выходе — готовый HTML-файл, который можно открыть в браузере без установки дополнительных программ.

---

## `pdf-to-md` — конвертация документов в Markdown

Преобразует PDF, PPTX, DOCX, XLSX и изображения в чистый Markdown через библиотеку MarkItDown от Microsoft. Удобно когда нужно извлечь текст из документа для дальнейшей работы: загрузить в языковую модель, отредактировать или сохранить в читаемом виде. Устанавливается автоматически при первом использовании, не требует настройки.

---

## `img` — генерация изображений

Создание изображений по текстовому описанию через внешние API (kie.ai и аналоги). Используется для подготовки визуалов к презентациям, обложек и иллюстраций. Поддерживает несколько моделей генерации с разным стилем и качеством.

---

## `caveman` — режим лаконичности

Переключает стиль ответов в максимально сжатый формат: без вводных фраз, без воды, только суть. Полезен когда нужно быстро получить ответ без объяснений. Имеет несколько уровней интенсивности — от слегка сокращённого до предельно краткого. Отключается командой `stop caveman`.

---

## `deploy` — публикация проекта на GitHub и Vercel ⭐ доп. балл

Полный цикл от локального проекта до живого сайта. Покрывает инициализацию git-репозитория, создание репозитория на GitHub, настройку удалённого подключения, перенос переменных окружения в Vercel и автоматический деплой при каждом пуше. Включает работу с несколькими аккаунтами и токенами, а также решения типичных ошибок: конфликты runtime, проблемы с правами доступа, кэширование переменных окружения. Скилл разработан для получения дополнительного балла (оценка 5).

---

## `planner` — расписание из Google Таблицы в Apple Notes

Читает Google Таблицу с расписанием, извлекает ключевые события — что и когда — и автоматически создаёт структурированную заметку в приложении Заметки на Mac. Заметка организована по дням и часам. Не требует ручного копирования: достаточно дать ссылку на таблицу с открытым доступом.
