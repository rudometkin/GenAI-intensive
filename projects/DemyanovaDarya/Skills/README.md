# Skills — Личные инструменты с Claude Code

Скиллы, которые я использую в работе над Private.Education и смежными задачами.

---

## `pitch-deck` — создание презентаций для инвесторов и партнёров

**Триггер:** нужно подготовить презентацию проекта — для инвестора, акселератора, корпоративного партнёра или внутреннего использования  
**Что делает:** структурирует питч по фреймворку (проблема → решение → рынок → бизнес-модель → команда → план), помогает сформулировать боли аудитории словами клиента, проверяет логику повествования  
**Как использую:** даю контекст проекта + аудиторию → получаю структуру с тезисами для каждого слайда → итерирую формулировки до нужного уровня конкретности

---

## `market-research` — анализ рынка и конкурентов

**Триггер:** нужно быстро разобраться в рынке, понять размер (TAM/SAM/SOM), найти конкурентов и их слабые места  
**Что делает:** помогает сформулировать гипотезы о сегментах рынка, структурирует данные из открытых источников, строит сравнительные таблицы конкурентов  
**Как использую:** описываю продукт и аудиторию → получаю структуру анализа → заполняю данными из интервью и открытых источников

---

## `customer-interview` — подготовка и анализ глубинных интервью

**Триггер:** нужно провести интервью с потенциальным клиентом или проанализировать записи уже проведённых  
**Что делает:** формирует гайд для интервью по методологии Jobs-to-be-Done, помогает вычленить джобы и боли из расшифровок, переформулирует боли словами клиента  
**Как использую:** перед интервью — получаю гайд. После — вставляю расшифровку, получаю карту джобов и цитаты для AJTBD-документа
