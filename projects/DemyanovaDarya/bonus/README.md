# Bonus Skills

Скиллы для дополнительного балла. Покрывают полный цикл работы над продуктом — от поиска инструментов до деплоя.

---

## Полный цикл

```
/project-search → /interface-analysis → /spec → /build → /test → /deploy
```

---

### `project-search` — поиск и сравнение сервисов

Ищет не менее 20 инструментов или сервисов под запрос, фильтрует по активности за последний год, оценивает по критериям и рекомендует лучший вариант с обоснованием.

### `interface-analysis` — анализ логики работы продукта

Глубокий разбор продукта через его интерфейсы: пользовательские сценарии, экраны, API, структуры данных. Результат — понимание как устроен продукт изнутри без доступа к коду.

### `spec` — написание технического задания

Пишет структурированное ТЗ по запросу или результатам анализа. Покрывает цели, user stories, экраны, API-контракт, бизнес-логику, edge cases и критерии готовности.

### `build` — реализация ТЗ

Реализует техническое задание шаг за шагом: создаёт файлы, пишет код, следует соглашениям проекта, не выходит за рамки скоупа.

### `test` — тестирование реализации

Проверяет реализацию по критериям из ТЗ: статический анализ, юнит-тесты, ручная проверка API, прогон по user stories. Выдаёт отчёт с конкретными несоответствиями.

### `deploy` — публикация на GitHub + Vercel

Полный цикл публикации: git init, GitHub репозиторий, деплой на Vercel, переменные окружения, автодеплой при пуше.
