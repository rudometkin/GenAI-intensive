# Project Search

> **Скилл для дополнительного балла**

Ищет и сравнивает не менее 20 сервисов под конкретный запрос, фильтрует по дате (последний год), оценивает каждый по критериям и выдаёт ранжированный список с рекомендацией.

---

## Triggers

- "найди сервис для", "сравни инструменты для", "что лучше выбрать"
- "find the best tool for", "compare services"
- `/project-search`

---

## Входные данные

- Задача или требование (что нужно сделать)
- Ограничения: бюджет, язык, платформа, лицензия (опционально)
- Целевая аудитория (опционально)

---

## Алгоритм

### Шаг 1: Сбор кандидатов

Запустить серию поисковых запросов:

```
"{задача} best tools 2024 2025"
"{задача} alternatives comparison"
"{задача} top services site:producthunt.com OR site:g2.com OR site:reddit.com"
"best {категория} tools {год} review"
```

Собрать минимум 20 уникальных кандидатов. Отфильтровать те, у которых нет активности за последний год.

### Шаг 2: Оценка по критериям

Для каждого кандидата оценить от 1 до 5:

| Критерий | Вес |
|---|---|
| Соответствие задаче | 30% |
| Активное развитие (последний год) | 20% |
| Документация / простота старта | 20% |
| Цена / бесплатный tier | 15% |
| Отзывы сообщества | 15% |

### Шаг 3: Shortlist топ-5

Оставить 5 лучших, отсортировать по итоговому баллу. Для каждого:
- Ссылка на сайт
- Дата последнего обновления / релиза
- Плюсы и минусы (по 3)
- Итоговый балл

### Шаг 4: Рекомендация

Выбрать одного победителя и обосновать — почему именно он под конкретный запрос пользователя, а не просто по рейтингу.
