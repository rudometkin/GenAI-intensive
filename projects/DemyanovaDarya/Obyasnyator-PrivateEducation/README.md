# Объяснятор

Проект начинался как клон [Opennote Lite](https://github.com/tommy-gg/opennote) — инструмента для работы с документами через языковую модель. В процессе адаптации под конкретную задачу функциональность была переработана настолько, что от исходного кода осталась только базовая архитектура.

**Объяснятор** — веб-приложение, которое превращает загруженный учебный документ в интерактивную учебную среду. Языковая модель работает исключительно в рамках загруженного текста, не привлекая внешние источники.

---

## Контекст разработки

Приложение создано в рамках проекта **Private.Education** — сервиса персонального образовательного сопровождения школьников. Проект занимается построением индивидуальных учебных треков: от профориентации до поступления в университет.

Одна из системных проблем в этой модели — самостоятельная работа ученика с материалами между занятиями с куратором. Без инструмента поддержки непонятые фрагменты остаются без разбора до следующей встречи, что замедляет прогресс. Объяснятор закрывает этот разрыв: ученик может задать вопрос, пройти проверку или повторить термины в любое время — и всё это строго по своему документу.

---

## Что добавлено относительно исходного проекта

Оригинальный Opennote Lite реализовывал загрузку документа и базовый чат. В рамках данного проекта добавлено:

- квиз с автоматической генерацией вопросов и разбором ошибок
- флеш-карточки с CSS-анимацией переворота
- практические задачи с прогрессивными подсказками
- функция объяснения выделенного фрагмента текста
- сохранение и экспорт заметок в формате Markdown
- русскоязычный интерфейс и адаптация логики под учебный процесс
