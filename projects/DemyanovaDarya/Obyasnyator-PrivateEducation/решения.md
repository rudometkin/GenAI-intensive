## Поиск по документу без базы данных

**Задача:** находить фрагменты документа, наиболее близкие к вопросу пользователя, и передавать их модели в качестве контекста.

**Решение:** поиск по ключевым словам. Для каждого фрагмента документа считается, сколько раз слова из вопроса встречаются в его тексте. Отбираются четыре фрагмента с наибольшим совпадением, их порядок в исходном документе восстанавливается.

```ts
function scoreChunk(chunk, query) {
  const words = query.toLowerCase().split(/\W+/).filter(w => w.length > 3);
  return words.reduce((score, word) => {
    return score + (chunk.text.toLowerCase().match(new RegExp(word, "g")) ?? []).length;
  }, 0);
}
```

**Почему не семантический поиск:** надёжнее для первой версии, не требует никакой дополнительной инфраструктуры, работает сразу.

## Получение структурированного ответа от модели

**Задача:** получить валидный массив данных для квиза или карточек без лишних пояснений и обёрток.

**Решение:** явно указываем в запросе к модели: «Верни только массив в формате JSON, без разметки». На случай если модель всё равно добавит оформление — очищаем ответ функцией `parseJSON()`.

```ts
export function parseJSON<T>(raw: string): T {
  const stripped = raw
    .replace(/^```(?:json)?\s*/i, "")
    .replace(/\s*```\s*$/, "")
    .trim();
  return JSON.parse(stripped) as T;
}
```

## Чтение PDF-файлов в Next.js

**Задача:** читать PDF-файлы на сервере без конфликтов со средой выполнения.

**Решение:** три обязательных шага:

```ts
// next.config.ts — исключаем библиотеку из серверной сборки
serverExternalPackages: ["pdf-parse"]

// route.ts — указываем среду выполнения
export const runtime = "nodejs";

// Динамический импорт вместо статического
const pdfParse = (await import("pdf-parse")).default;
```

## Анимация переворота карточки без библиотек

**Задача:** реализовать эффект переворота флеш-карточки при клике.

**Решение:** стандартные возможности CSS — перспектива, трёхмерное вращение и скрытие обратной стороны элемента.

```tsx
<div style={{ perspective: "1000px", height: "220px" }}>
  <div style={{
    transformStyle: "preserve-3d",
    transform: flipped ? "rotateY(180deg)" : "rotateY(0deg)",
    transition: "transform 0.5s"
  }}>
    <div style={{ backfaceVisibility: "hidden" }}>/* Лицевая сторона */</div>
    <div style={{ backfaceVisibility: "hidden", transform: "rotateY(180deg)" }}>/* Обратная сторона */</div>
  </div>
</div>
```

**Задача:** опубликовать приложение с библиотекой чтения PDF, которая требует стандартной серверной среды Node.js.

**Решение — три обязательных условия:**

2. Указать `export const runtime = "nodejs"` в файле маршрута загрузки
3. Использовать динамический импорт: `const pdfParse = (await import("pdf-parse")).default`

## Подсветка активного раздела в навигации

**Задача:** подсвечивать в боковой панели тот раздел документа, который сейчас виден на экране.

**Решение:** наблюдатель пересечений (`IntersectionObserver`) следит за каждым разделом документа с порогом видимости 30%. При клике по разделу в боковой панели документ плавно прокручивается к нему.
