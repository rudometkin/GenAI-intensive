# Объяснятор — User Scenarios & Target Audience

## Целевая аудитория

**Фокус V1 — две группы:**

1. **Студенты** — загружают лекции, учебники, слайды с курса; хотят быстро разобраться с непонятным материалом перед экзаменом
2. **Самообучающиеся** — проходят онлайн-курсы, читают статьи/книги; хотят структурировать знания и проверить себя

**За пределами V1 (не трогать):**
- Школы и преподаватели
- Collaborative learning / "invite friends"
- Researchers (нужна более глубокая интеграция с источниками)
- Community features, public publishing

---

## Главный пользовательский сценарий

```
1. Пользователь открывает сайт
2. Загружает PDF лекции или конспект
3. Система извлекает текст и строит рабочее пространство
4. Пользователь нажимает "Explain this chapter"
5. AI tutor объясняет содержание по документу
6. Пользователь нажимает "Quiz me"
7. Система задаёт вопросы, ждёт ответ, даёт feedback
8. Пользователь генерирует flashcards
9. Сохраняет итоговые study notes
```

---

## Дополнительные сценарии

### Сценарий: "Не понимаю этот абзац"
1. Выделяет текст в документе
2. Нажимает "Explain this"
3. Tutor объясняет выделенное простыми словами со ссылкой на контекст документа

### Сценарий: "Хочу проверить себя"
1. Нажимает "Quiz me"
2. Tutor задаёт вопросы один за другим
3. После каждого ответа — feedback и объяснение правильного ответа

### Сценарий: "Готовлюсь к экзамену"
1. Генерирует flashcards из документа
2. Тренируется на карточках
3. Сохраняет слабые места в Study Notes
4. Просит tutor объяснить слабые темы подробнее

### Сценарий: "Хочу структурировать материал"
1. Нажимает "Explain visually"
2. Получает concept map или structured outline
3. Сохраняет в Study Notes
