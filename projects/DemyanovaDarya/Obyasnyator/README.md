# Объяснятор

**Объяснятор** — ИИ-тьютор для самостоятельного изучения любых материалов. Загружаешь документ, получаешь персонального преподавателя внутри него.

Проект вдохновлён [Opennote Lite](https://github.com/tommy-gg/opennote) — инструментом для работы с документами через AI. Объяснятор — его русскоязычная версия, адаптированная под задачи образовательного сопровождения.

---

## Зачем это нужно

В рамках проекта **Private.Education** — сервиса персонального образовательного сопровождения школьников — одной из ключевых задач стала работа с учебными материалами: лекциями, конспектами, учебниками. Объяснятор решает эту задачу: ученик загружает документ и может задавать вопросы, проходить квизы и работать с карточками — всё строго по материалу, без лишней информации из интернета.

---

## Что умеет

### Загрузка материалов
- PDF, TXT, Markdown
- Вставка текста напрямую
- Автоматическое разбиение на разделы

### Чат с тьютором
- Отвечает только по загруженному документу
- Объясняет сложные места простым языком
- При выделении текста — кнопка «Объяснить»
- Отвечает на языке пользователя

### Активное обучение
- **Квиз** — 5 вопросов по материалу с вариантами ответов и объяснениями
- **Карточки** — флеш-карточки с понятиями и определениями, с анимацией переворота
- **Практика** — задачи с подсказками и решением

### Заметки
- Сохранение ответов тьютора одной кнопкой
- Экспорт всех заметок в Markdown-файл

### Навигация
- Боковая панель с разделами документа
- Подсветка активного раздела при прокрутке

---

## Как использовать

1. Открой [объяснятор на Vercel](https://obyasnyator.vercel.app)
2. Загрузи учебный материал или нажми «Демо»
3. Задавай вопросы, проходи квиз, учи карточки

## Технологии

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Claude AI (Anthropic) — `claude-sonnet-4-6` для чата, `claude-haiku-4-5-20251001` для генерации
- pdf-parse для обработки PDF

## Связь с Private.Education

**Private.Education** — сервис комплексного образовательного сопровождения школьников: от профориентации до поступления в вуз. Индивидуальный трек, преподаватели из ведущих вузов (МФТИ, МГУ, ВШЭ), психологическая поддержка.

Объяснятор — технологический инструмент внутри этой системы: помогает ученику самостоятельно работать с учебными материалами между занятиями с куратором.

## Навыки, необходимые для разработки

### Разработка
- **TypeScript** — строгая типизация всего проекта, интерфейсы для всех структур данных
- **React / Next.js** — компонентный подход, серверные маршруты, App Router
- **Tailwind CSS** — стилизация без написания CSS вручную
- **REST API** — проектирование серверных маршрутов для загрузки, чата, генерации

### Работа с ИИ
- **Промпт-инжиниринг** — составление системных промптов, управление контекстом модели
- **Anthropic API** — интеграция языковых моделей, выбор модели под задачу
- **Структурированный вывод** — получение валидного JSON от языковой модели

### Алгоритмы и данные
- **Обработка текста** — разбиение документа на смысловые фрагменты
- **Поиск по ключевым словам** — ранжирование фрагментов по релевантности без векторных баз данных
- **Работа с PDF** — извлечение текста из документов на сервере

### Интерфейс
- **CSS-анимации** — трёхмерный переворот карточек средствами чистого CSS
- **Browser API** — работа с выделением текста, хранилищем браузера, наблюдателем пересечений
- **Адаптивная вёрстка** — трёхколоночный интерфейс, модальные окна, уведомления

### Инфраструктура
- **Git / GitHub** — контроль версий, работа с ветками
- **Vercel** — публикация Next.js приложения, переменные окружения
- **Node.js** — понимание серверной среды выполнения и её ограничений
