# PERSONAL

Персональная AI-операционная система product manager'а. Набор skills под реальные рабочие задачи: growth, analytics, stakeholder management, продуктовые запуски и собственные проекты в области elder care / care economy.

---

## 1. Используемые personal AI-skills

### 🔬 Продуктовый анализ и исследования

**`jtbd-extractor`** — когда нужно разобрать незнакомый продукт или сформулировать core scenario для MVP — извлекает главный Job-to-be-Done, строит user flow, отделяет must-have от nice-to-have — подкладывает описание продукта, целевую аудиторию, контекст рынка

**`product-architecture-mapper`** — когда нужна декомпозиция продукта на модули или оценка scope — раскладывает на 4–7 независимых модулей, строит матрицу модуль→навык, предлагает 3 варианта scope — подкладывает описание продукта, user flow, технический стек

**`cloneability-assessment`** — когда нужно оценить сервис для клонирования, анализа или конкурентного ресёрча — оценивает по 10 критериям (1–5), выдаёт честный вердикт и сужение scope — подкладывает название сервиса, целевую аудиторию, учебный/бизнес-контекст

**`conversion-funnel-analyzer`** — когда падает конверсия или нужно найти точку потерь в воронке — диагностирует шаги, находит drop-off, предлагает гипотезы причин и следующие шаги — подкладывает данные по шагам воронки, сегменты, текущие метрики

**`research-synthesizer`** — когда накопились интервью, отзывы или NPS-комментарии — синтезирует в инсайты, находит паттерны, формулирует jobs и боли — подкладывает сырые данные ресёрча, контекст аудитории, цель исследования

---

### 📈 Growth и эксперименты

**`growth-experiment-designer`** — когда нужно оформить гипотезу в A/B тест или продуктовый эксперимент — генерирует план эксперимента: метрика, control/test, размер выборки, критерии успеха, риски — подкладывает гипотезу, текущие метрики, трафик, продуктовый контекст

**`ab-test-analyzer`** — когда получены результаты теста и нужен вывод — интерпретирует статзначимость, строит вывод, даёт рекомендацию по запуску или повтору — подкладывает данные теста, метрику, p-value или сырые числа

**`moat-designer`** — когда нужно оценить устойчивость продукта или сформулировать конкурентный ров — анализирует 7 источников moat, оценивает силу каждого, предлагает стратегию укрепления — подкладывает описание продукта, рынок, конкурентов, бизнес-модель

---

### 📣 Коммуникация и документы

**`stakeholder-brief-writer`** — когда нужно подготовить апдейт для C-level, синхронизацию с командой или запрос на ресурсы — генерирует структурированный бриф: context → decision needed → options → recommendation — подкладывает тему, аудиторию, желаемый outcome, ключевые факты

**`prd-writer`** — когда нужен PRD, one-pager или product spec для новой фичи — создаёт документ: проблема, решение, метрики успеха, scope, edge cases, open questions — подкладывает описание фичи, целевую аудиторию, бизнес-цель, технические ограничения

---

### 🚀 Запуск продуктов и стартапы

**`frontend-mvp-builder`** — когда нужен работающий прототип для демо или валидации идеи — генерирует Next.js + Tailwind фронтенд с мокданными и ключевым user flow — подкладывает описание продукта, модули, дизайн-референсы, технический стек

**`startup-pitch-builder`** — когда нужен pitch deck, investment memo или one-pager для нового проекта — строит нарратив: проблема → рынок → решение → traction → ask — подкладывает описание проекта, рынок, команду, метрики, контекст встречи

---

### 🌿 Нишевые проекты

**`elder-care-researcher`** — когда нужен ресёрч рынка elder care, серебряной экономики или смежных тем — собирает данные по рынку, сегментам, барьерам, конкурентам, регуляторике, трендам — подкладывает гео, сегмент аудитории, тип исследовательского вопроса, глубину

## 2. Задачи, под которые планирую собрать дополнительные skills

### Аналитика и данные
- **SQL-ассистент для продуктовых метрик** — генерация запросов по описанию метрики (DAU, retention, funnel) под мою схему данных
- **Дашборд-дизайнер** — проектирование структуры дашборда из набора метрик и целевой аудитории
- **Когортный аналитик** — интерпретация когортного анализа, выявление retention-паттернов

### Продуктовая стратегия
- **Генератор продуктовой стратегии** — из цели бизнеса + контекста рынка → roadmap-логика, OKR, приоритеты
- **Оценщик гипотез ICE/RICE** — автоматическая расстановка приоритетов с обоснованием

### Карьера и личный бренд
- **PM-карьерный пакет** — резюме, LinkedIn, кейс-интерпретация под конкретную роль и компанию
- **Кейс-интервью тренер** — симуляция product case interview с обратной связью

### Коммуникации и переговоры
- **C-level коммуникатор** — подготовка к сложным переговорам, executive summary, управление ожиданиями
- **OKR-фасилитатор** — формулировка OKR, выравнивание с командой, трекинг

### Проекты в elder care / care economy
- **High-trust UX designer** — UX-паттерны для уязвимых аудиторий: пожилые, горе, стресс, медицина
- **Care economy market mapper** — систематический ресёрч смежных рынков: elder care, palliative, grief tech
- **Regulatory navigator** — анализ регуляторных барьеров для продуктов в здравоохранении

### AI-assisted execution
- **Нарратив-дизайнер** — построение сильного нарратива для презентаций, статей, питчей
- **Встреча-ассистент** — подготовка к встречам: пре-рид, вопросы, follow-up action items
