# Service Search for Clone

## Статус
Используется сейчас

## Когда триггерится
- Нужно найти реальные сервисы в заданной теме или рынке
- Хочу посмотреть, что уже существует в нише перед запуском продукта
- Формирую шорт-лист конкурентов или кандидатов для анализа

**Фразы:** «найди сервисы для клонирования в теме X», «подбери кандидатов для учебного клона», «какие сервисы есть на рынке Y», «что существует в нише»

## Что делает
Ищет реальные существующие сервисы через WebSearch + WebFetch. Только то, что действительно работает — без придуманных примеров. Группирует по категориям, даёт краткое описание и первичную оценку каждого кандидата.

## Входные данные
- Тема или рынок (обязательно) — например: «серебряная экономика», «elder care», «death admin»
- Количество кандидатов (опционально, дефолт: 15–20)
- Гео-фокус (опционально)
- Исключения (опционально)

## Что выдаёт на выходе
Структурированный список сервисов, разбитых по категориям: название, страна, краткое описание, почему интересен для анализа.

## Как помогает мне в работе
Ускоряет конкурентный ресёрч с нескольких часов до 15 минут. Полезен как при учёбе, так и в реальной продуктовой работе перед запуском в новой нише.

## Пример реального использования
По теме «серебряная экономика» нашёл 20 реальных сервисов (CareZone, Papa, Settld, GoodTrust и др.) — они стали базой для cloneability assessment.
