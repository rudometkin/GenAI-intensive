# Новый Скилл (Pattern → Skill Converter)

## Статус
Используется сейчас

## Когда триггерится
- Только что выполнил сложную многошаговую задачу — хочу сохранить паттерн
- Есть повторяющийся рабочий workflow, который хочу упаковать в skill
- Хочу создать персональный skill под специфическую задачу

**Фразы:** «сделай skill из этого», «упакуй в skill», «save this as a skill», «хочу переиспользовать этот воркфлоу», `/новый-скилл`

## Что делает
1. Анализирует текущий разговор или описание пользователя
2. Извлекает паттерн: инструменты, последовательность шагов, форматы входа/выхода
3. Проверяет: не дублирует ли существующие skills
4. Генерирует готовый `SKILL.md` с правильной структурой
5. Устанавливает skill в `~/.claude/skills/`
6. Предлагает тест для проверки

## Входные данные
- Описание паттерна или команда `/новый-скилл` после завершения задачи
- Название skill (опционально — предложит само)
- Контекст: для чего skill нужен (опционально)

## Что выдаёт на выходе
Готовый `SKILL.md`, установленный в систему, проверенный на корректность. Skill сразу доступен для использования.

## Как помогает мне в работе
Мета-инструмент: позволяет наращивать собственную AI операционную систему. Любой удачный workflow превращается в переиспользуемый skill за 5 минут.

## Пример реального использования
После сессии с созданием документации PROJECTCLONE: мог бы упаковать паттерн «анализ продукта → декомпозиция → MVP → деплой» в единый orchestration skill.
