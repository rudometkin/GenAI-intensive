# PushToGenAI Public

## Статус
Используется сейчас

## Когда триггерится
- Нужно опубликовать материалы курса в общий репозиторий GenAI Intensive
- Создан новый файл или документ — нужно закоммитить и запушить
- Завершена часть домашнего задания — нужна публикация

**Фразы:** «запушь в репозиторий», «опубликуй в GenAI», «сохрани в репо», `/PushToGenAI_public`

## Что делает
1. Проверяет, что файл готов к публикации (нет секретов, персональных данных, запрещённых паттернов)
2. Добавляет файл в нужную директорию репозитория
3. Создаёт коммит с осмысленным сообщением
4. Пушит в `rudometkin/GenAI-intensive` на GitHub
5. Возвращает подтверждение и путь к файлу

## Входные данные
- Контент для публикации (файл или текст)
- Целевой путь в репозитории
- Сообщение коммита (опционально — сгенерирует само)

## Что выдаёт на выходе
Подтверждение успешного пуша, путь к файлу в репозитории, ссылка на GitHub.

## Как помогает мне в работе
Автоматизирует рутину публикации учебных материалов. Не нужно переключаться в терминал или помнить git-команды.

## Пример реального использования
Запушил LOGIKA_VIBORA_SERVISA.md, PODGOTOVKA_K_RAZRABOTKE.md, README.md и код MVP — всё из одного разговора без ручных git-команд.
