# kbzhuy-start

**Триггер:** начало рабочей сессии по проекту КБЖУЙ

## Что делает

1. Читает `kbzhuy/docs/tracker.md` — что уже готово
2. Читает `kbzhuy/docs/implementation_plan.md` — текущая фаза и следующий шаг
3. Кратко перечисляет: что сделано, что в работе, что заблокировано
4. Предлагает одну конкретную задачу на эту сессию

## Контекст

```
Читать перед ответом:
- kbzhuy/docs/tracker.md
- kbzhuy/docs/implementation_plan.md
- kbzhuy/docs/decisions/log.md (последние 5 записей)
```

## Выход

```
## Статус: Фаза [N]

**Готово:** ...
**В работе:** ...
**Заблокировано:** ...

**Предлагаю на эту сессию:** [одна задача]
```
