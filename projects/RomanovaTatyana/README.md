# Навыки работы с кодом Claude

## Обзор

Этот репозиторий содержит структурированный набор навыков многократного использования, используемых в
Claude Code для поддержки рабочих процессов разработки стартапов.\
Каждый навык определяется условиями запуска, поведением и введенным
контекстом.

------------------------------------------------------------------------

## Определения навыков

### ИдеяВалидатор

- Триггер: появляется идея нового продукта или функции
- Действие: Оценивает жизнеспособность идеи на рынке, среди потребителей и
конкурентов
- Контекст:
    - Описание идеи
    - Целевая аудитория
    - Постановка проблемы
    - Охват рынка

### LeanCanvasGenerator

- Триггер: Необходимость структурировать бизнес-идею
- Действие: Создание компонентов Lean Canvas
- Контекст:
    - Описание стартапа
    - Ограничения (ресурсы, время, рынок)
    - Проверенные предположения

### MVPScopeBuilder

- Триггер: Переход от идеи к реализации
- Действие: Определяет минимальную жизнеспособную область применения продукта
- Контекст:
    - Lean Canvas
    - Гипотезы о ценности
    - Пользовательские сценарии

### UserPersonaCreator

- Триггер: Необходимо понимать целевых пользователей
- Действие: Создает образы пользователей
- Контекст:
    - Сегменты аудитории
    - Болевые точки
    - Поведенческие модели

### CustomerJourneyMapper

- Триггер: требуется дизайн UX или воронки
- Действие: Отображает путь клиента от начала до конца
- Контекст:
    - Персонажи
    - Точки соприкосновения
    - Взаимодействие с продуктом

### GrowthExperimentDesigner

- Триггер: Необходимость проверки гипотез о росте
- Действие: Разработка экспериментов и A/ B-тестов
- Контекст:
    - Показатели воронки
    - Гипотезы о росте
    - Текущие данные о производительности

------------------------------------------------------------------------

## Примечания

- Навыки являются модульными и составными\
- Качество контекста напрямую влияет на качество продукции\
- Эта система постоянно развивается в соответствии с потребностями разработки продукта
