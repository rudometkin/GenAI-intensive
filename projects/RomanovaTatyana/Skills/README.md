## kopirka

Скилл для работы над проектом **Копирка** — инструментом конвертации URL в чистый markdown, JSON, ссылки и summary.

Использовать при разработке: frontend, backend, логика извлечения контента, QA.

**Файлы:**
- `SKILL.md` — описание и инструкции скилла
- `references/architecture.md` — архитектура проекта
- `references/product-scope.md` — границы продукта
- `references/ui-rules.md` — правила UI
