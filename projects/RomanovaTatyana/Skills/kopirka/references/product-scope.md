# Product Scope: Копирка

## Что такое Копирка
Копирка — веб-приложение, которое превращает любую страницу по URL в чистый markdown, JSON и summary для дальнейшей работы с AI.

> "Turn any URL into clean, structured, AI-ready content in one click."

Вдохновлён core use case Firecrawl: scrape любого URL в clean data. Намеренно ограничен до одной функции — URL → clean output.

---

## Проблема
Обычная веб-страница неудобна для LLM и последующей обработки. Сырой HTML содержит мусор: меню, cookie banners, скрипты, стили, лишние блоки.

**Боль пользователя:**
- "Мне нужен чистый текст страницы, а не HTML-мусор."
- "Мне нужен markdown/JSON, чтобы потом отправить в LLM, RAG или сохранить."
- "Я не хочу сам разбираться с JS-сайтами, блокировками, динамическим контентом."
- "Я хочу вставить ссылку и сразу получить результат."

---

## Целевая аудитория
- AI/LLM разработчики
- Люди, строящие RAG-пайплайны
- Product managers / аналитики
- Контент-исследователи
- Студенты и indie hackers, которым нужно быстро "снять" содержимое сайта

---

## Сценарии использования

### Главный сценарий
1. Пользователь открывает сайт
2. Вставляет URL
3. Нажимает Convert
4. Через 1–3 секунды видит: Markdown, JSON, Summary, Links
5. Копирует нужный результат или скачивает файл

### Альтернативный сценарий
Пользователь вставляет URL статьи или документации → выбирает режим Summary only → получает короткую выжимку для чтения.

### Продвинутый сценарий (V2)
Пользователь вставляет домен → выбирает Map site или Crawl site → получает список страниц и может выбрать, какие конвертировать.

## MVP — что делаем

| Фича | Описание |
|------|----------|
| URL input | Поле для вставки ссылки + кнопка Convert |
| Вкладка Markdown | Чистый markdown: заголовки, абзацы, списки, таблицы, ссылки |
| Вкладка JSON | Структурированный JSON с url, title, content, headings, links |
| Вкладка Summary | 5–7 тезисов + 1 абзац summary |
| Вкладка Links | Список всех найденных ссылок на странице |
| Copy / Download | Кнопки для каждого формата (.md, .json) |
| История запросов | Последние 5–10 обработанных URL |
| JS fallback | Playwright для JS-rendered страниц |

---

## MVP — что НЕ делаем
- Полный crawl всего сайта
- Поиск по интернету
- Agent-like extraction
- Browser automation для пользователя
- Авторизация, команды, биллинг
- Сложные anti-bot обходы
- Enterprise-инфраструктура

---

## V2 — возможные расширения

1. **Extract links** — показать все найденные ссылки на странице отдельной вкладкой
2. **Crawl domain** — пользователь вставляет домен, получает список страниц + markdown по каждой
3. **Map site** — показать URL-структуру всего сайта
4. **Schema extraction** — извлечение по заданной схеме (title, author, publish date, price, faq items)
