# SKILL: Computer Vision Damage Assessment (CV-DA)

## 1. ОБЛАСТЬ ПРИМЕНЕНИЯ
Данный модуль предназначен для анализа изображений транспортных средств с целью выявления механических повреждений, коррозии и нарушения геометрии.

## 2. КЛАССИФИКАЦИЯ ДЕФЕКТОВ (Детекция и Сегментация)
Используйте архитектуры типа **Mask R-CNN** или **YOLOv8-seg** для локализации следующих классов:
* **Scratch (Царапина):** Линейное нарушение ЛКП без деформации металла.
* **Dent (Вмятина):** Деформация поверхности с изменением бликов.
* **Crack (Трещина):** Сквозное или глубокое повреждение пластиковых элементов (бамперы, фары).
* **Corrosion (Коррозия):** Очаги окисления, классифицируемые по площади.

## 3. МАТЕМАТИЧЕСКАЯ МОДЕЛЬ ОЦЕНКИ
Для расчета стоимости ремонта используется коэффициент тяжести $S$:
$$S = \sum_{i=1}^{n} (A_i \cdot K_{material} \cdot D_{depth})$$
Где:
- $A_i$: Площадь поврежденного сегмента в $px^2$ (пересчитанная в $cm^2$ через эталонный объект, например, госномер).
- $K_{material}$: Коэффициент материала (алюминий > сталь > пластик).
- $D_{depth}$: Глубина деформации (определяется по искажению отраженных паттернов).

## 4. ПРАВИЛА ВЫВОДА
1. Выделить маской зону повреждения.
2. Определить тип детали (крыло, дверь, капот).
3. Сопоставить с нормо-часами из базы данных автосервиса.
