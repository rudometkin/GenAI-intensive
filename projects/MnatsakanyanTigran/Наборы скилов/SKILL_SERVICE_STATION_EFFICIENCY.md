# SKILL: E-commerce Logistics & Avito Automation

## 1. СТРУКТУРА ДАННЫХ ДЛЯ МАССПOСТИНГА
Каждое объявление генерируется на основе матрицы совместимости запчасти.

## 2. ГЕНЕРАТОР ЗАГОЛОВКОВ (SEO-Friendly)
Шаблон: `[Деталь] + [Марка/Модель] + [Год] + [Особенность]`
> *Пример: "Компрессор пневмоподвески Mercedes-Benz W221 (2009) Восстановленный"*

## 3. ФИНАНСОВЫЙ МОНИТОРИНГ
Динамическое изменение цены в Excel-таблице на основе:
- Цен конкурентов на Авито (Parsing).
- Срочности освобождения склада.
- Количества просмотров за последние 24 часа.

## 4. СТРАТЕГИЯ ДУБЛЕЙ
Использование различных фонов для фото через **Stable Diffusion (Inpainting)** для обхода фильтров Авито на одинаковый товар.
