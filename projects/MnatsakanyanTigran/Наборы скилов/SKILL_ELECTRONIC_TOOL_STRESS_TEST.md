# SKILL: Industrial OCR for Automotive ID

## 1. ТЕХНИЧЕСКИЙ КОНТЕКСТ
Распознавание символов на грязных, замасленных или выштампованных поверхностях (металл).

## 2. АЛГОРИТМ ПРЕПРОЦЕССИНГА
Перед подачей в OCR-движок (Tesseract/PaddleOCR), необходимо применить:
1.  **CLAHE** (Contrast Limited Adaptive Histogram Equalization) для выравнивания освещения.
2.  **Adaptive Thresholding** для выделения выштамповки.
3.  **Perspective Correction** на основе детекции углов таблички.

## 3. ВАЛИДАЦИЯ VIN
Алгоритм проверки контрольной суммы (9-й символ для рынка США):
- Присвоение весов буквам (A=1, B=2...).
- Расчет остатка от деления на 11.

## 4. ЛОГИКА СОВМЕСТИМОСТИ
При успешном распознавании VIN, выполнить запрос к API каталогов (TecDoc, оригинальные каталоги) для получения взрыв-схем (Exploded View).
