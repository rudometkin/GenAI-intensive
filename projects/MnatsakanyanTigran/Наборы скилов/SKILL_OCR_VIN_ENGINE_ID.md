# SKILL: Predictive Maintenance for Power Tools

## 1. АНАЛИЗ ЗВУКОВОГО СПЕКТРА
Использование БПФ (Быстрое преобразование Фурье) для анализа шума двигателя.
- Пики на частотах $X$: Износ подшипников.
- Искрение на коллекторе: Анализ спектра свечения (CV) или пульсации тока.

## 2. ТЕПЛОВАЯ КАРТА (Thermal Imaging)
Интеграция с тепловизорами для поиска зон перегрева в обмотках статора.

## 3. СТРАТЕГИЯ REURBISHED
Алгоритм принятия решения: Ремонт vs Списание на запчасти (на основе стоимости запчастей и остаточного ресурса АКБ).
