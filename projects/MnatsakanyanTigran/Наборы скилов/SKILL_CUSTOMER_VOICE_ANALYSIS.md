# SKILL: Computer Vision for Wheel Alignment (3D-Сход-Развал)

## 1. ДЕТЕКЦИЯ МИШЕНЕЙ
Распознавание шахматных паттернов (мишеней) на колесах. Использование функции `cv2.findChessboardCorners()` для определения вектора нормали колеса относительно камеры.

## 2. ГЕОМЕТРИЧЕСКИЙ РАСЧЕТ
Вычисление углов:
- **Camber (Развал)**
- **Toe (Схождение)**
- **Caster (Кастор)**

## 3. ВИЗУАЛИЗАЦИЯ ДЛЯ МАСТЕРА
Наложение AR-элементов (стрелок) на видеопоток, указывающих направление вращения регулировочных болтов.
