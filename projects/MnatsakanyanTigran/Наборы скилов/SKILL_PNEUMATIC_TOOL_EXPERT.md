## 2. ВИЗУАЛЬНОЕ СРАВНЕНИЕ (Siamese Networks)
Использование сиамских сетей для сравнения снятой детали (б/у) с эталонным фото новой детали из каталога для подтверждения совместимости.

## 3. ПРОВЕРКА КОМПЛЕКТНОСТИ
