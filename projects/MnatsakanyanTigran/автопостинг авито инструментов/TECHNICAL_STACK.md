# Техническая документация ToolDrive

## База данных
- Используется SQLite или PostgreSQL для хранения остатков и истории публикаций.

## Модуль Авито
- Интеграция через Excel-автозагрузку (Автокабинет).
- Поддержка уникализации изображений (наложение водяных знаков или изменение метаданных).

## Модуль аналитики
- Расчет ROI по каждому объявлению на основе затрат на продвижение и маржи.
