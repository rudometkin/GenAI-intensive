# Переход на Avito API

Когда объем превысит 100 объявлений в день, рекомендуется перейти с Excel на Direct API:
1. **POST /items**: Создание объявления.
2. **GET /items/{id}/stats**: Получение статистики просмотров и контактов.
3. **Webhook**: Мгновенное получение сообщений из чата и ответ через ИИ (ChatGPT API).
