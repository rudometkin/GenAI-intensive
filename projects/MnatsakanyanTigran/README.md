## 1. ASMLabs — Enterprise AI Agent Hub

**Папка:** `ASMLabs/`

Многоагентная система для автосервисного бизнеса. Объединяет компьютерное зрение, NLP и CRM-логику в единый пайплайн: диагностика автомобиля по изображению → оценка стоимости ремонта → коммуникация с клиентом в стиле мастера.

| Файл | Содержимое |
|---|---|
| `Enterprise AI Agent Hub.py` | Ядро системы: оркестрация агентов, CV-модуль (OpenCV), обработка аудио-транскриптов |
| `AVITO_CLONE_LOGIC.md` | Протокол CRM-агента: тон общения, маппинг слэнга, правила ценообразования |
| `SPEC_WRITER.md` | Инструкции AI Technical Writer: стиль документации, метрики точности, hardware-спецификации |
| `TECH_ADAPT.md` | Адаптация технического стека под конкретные задачи автосервиса |

**Стек:** Python, OpenCV, PyTorch, Flask, pandas, dataclasses

## 2. Автопостинг Авито — ToolDrive

**Папка:** `автопостинг авито инструментов/`

Система автоматизации публикаций объявлений для ресейл-бизнеса в нише инструментов (пневматика, промышленное оборудование). Генерирует объявления, уникализирует контент, считает ROI по каждой публикации.

| Файл | Содержимое |
|---|---|
| `AvitoAutomation.py` | Генерация CSV/Excel для автозагрузки в Авито Кабинет, уникализация описаний |
| `STRATEGY.md` | Стратегия ниши ToolDrive: ресейл + агентская схема, сегменты пневматика/электроинструмент |
| `API_INTEGRATION.md` | Интеграция с Авито API и Excel-автозагрузкой |
| `CONTENT_GUIDE.md` | Правила генерации текстов объявлений: структура, ключевые слова, уникализация |
| `MARKET_ANALYSIS.md` | Анализ рынка: конкуренты, ценовые диапазоны, сезонность |
| `TECHNICAL_STACK.md` | Архитектура: SQLite/PostgreSQL, модуль аналитики, расчёт ROI |

**Стек:** Python, pandas, SQLite/PostgreSQL, Авито API (Excel-автозагрузка)
