# PRODUCT -- О продукте

## Агрегаторы нейросетей

- **Claude API (Anthropic):** Haiku 4.5 -- основная модель для PoC (скорость, ~200 руб/мес/чел). Sonnet 4.5 -- для сравнения качества (A/B тест).
- **Gemini (Google, планируется):** аналитика конверсии ответов, оптимизация промптов на данных.

## Парсеры

- **Profi.ru scraping:** 800+ профилей репетиторов собраны (предметы, ставки, отзывы, стиль описания)
- **Notification parser (Android):** NotificationListenerService перехватывает push от Профи.ру, парсит предмет, уровень, текст запроса, бюджет
- **Funnel data:** воронка по предметам/ставкам -- при какой цене/предмете конверсия выше

## Продукт и целевая аудитория

**Продукт:** AI-ассистент, который автоматически откликается на заявки учеников на Профи.ру, ведет первичную переписку, берет контакт и конвертирует в лида.

**Целевая аудитория (5 AJTBD-сегментов):**

1. Загруженный профи (20-30 занятий/нед, 4-8ч/день на уроках, некогда откликаться)
2. Репетитор-предприниматель (2-5 профилей, 40-60 откликов/день, нанимает ассистентов)
3. Мама в декрете (2-4ч рабочие окна, телефон в руках но внимание разорвано)
4. Начинающий (0-10 отзывов, конверсия 5-10%, нужно 100-200 откликов на 10 учеников)
5. По совместительству (учитель/программист, не хочет заниматься продажами)

**Боль:** Репетиторы недозарабатывают из-за неэффективного поиска учеников. 4-8 часов/день на уроках -- некогда мониторить заявки. Средний ученик уходит через 2-4 месяца -- нужен постоянный поток новых.

**Как работает:** Android-приложение перехватывает push-уведомления от Профи.ру, отправляет данные заказа на наш сервер, получает AI-сгенерированный ответ и автоматически заполняет форму отклика. Репетитор получает уведомление в Telegram и может подтвердить или отменить.

**Unit Economics:**

| Параметр | Значение |
|----------|----------|
| COGS на клиента | ~200 руб/мес (только LLM API) |
| Цена | 990 руб/мес фикс + 2500 руб/подтвержденный ученик |
| Средний чек (при ~3 учениках/мес) | ~8 500 руб |
| Маржа | ~97% |
| WTP (по CustDev) | 2-5K/мес у загруженных, 500-1500 у начинающих |
