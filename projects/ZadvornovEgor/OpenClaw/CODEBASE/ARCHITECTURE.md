> Код является приватным и не публикуется в открытом доступе.
> Доступ к исходному коду по запросу: **@egor_zadvornov** (Telegram)

---

## Архитектура системы

```
[Телефон репетитора]
  |
  | NotificationListenerService
  | -> перехват уведомления Профи.ру
  | -> парсинг заказа (предмет, класс, формат, бюджет)
  |
  | HTTP POST /generate-response
  v
[Наш сервер (FastAPI)]
  |
  | -> Claude API (генерация персонализированного ответа)
  | -> Telegram Bot API (уведомление репетитору)
  |
  | HTTP Response (текст ответа)
  v
[Телефон репетитора]
  |
  | Accessibility Service
  | -> открытие страницы заказа в приложении Профи.ру
  | -> заполнение формы ответа
  | -> отправка отклика
```

## Ключевые технические решения

1. **NotificationListenerService** -- системный сервис Android, не требует root. Перехватывает все push-уведомления и извлекает данные заказа.

2. **AccessibilityService** -- позволяет программно взаимодействовать с UI других приложений. Используется для автоматического заполнения формы отклика в приложении Профи.ру.

3. **Claude API через наш сервер** -- клиентское приложение не хранит API-ключи. Все запросы к LLM проксируются через FastAPI-сервер, что позволяет логировать, кэшировать и контролировать расходы.

4. **Telegram-бот** -- уведомляет репетитора о новых заказах и отправленных откликах. Позволяет подтвердить или отменить автоотклик до отправки (опционально).
