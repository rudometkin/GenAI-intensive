# Клон Gamma.app -- SlideDeck AI

> Анализ и клонирование Gamma.app через open-source решение SlideDeck AI.

---

## 1. Сервис

**Gamma.app** -- AI-платформа для создания презентаций. Пользователь вводит тему, AI генерирует структуру, контент и дизайн слайдов. Весь процесс: от текстового промпта до готовой презентации за минуты.

**Почему интересен:**

- Рынок презентаций оценивается в $16B+ (PowerPoint, Google Slides, Keynote, Canva). AI-генерация -- принципиально новый способ создания контента, который меняет workflow миллионов пользователей.
- Gamma привлекла $20M+ инвестиций, набрала 5M+ пользователей. Это подтверждение product-market fit.
- Простой интерфейс: тема --> настройки стиля --> структура --> слайды --> экспорт (PDF/PPTX/link). Барьер входа для пользователя минимален.
- Монетизация: freemium-модель с лимитами на генерации. Plus за $10/мес, Pro за $20/мес.

**Ключевая ценность:** пользователь экономит 2-4 часа на создании презентации. Вместо пустого слайда -- готовый драфт, который нужно только отредактировать.

## 3. Модули продукта

| Модуль | Описание | Как реализовано в Gamma | Как в SlideDeck AI |
|--------|----------|------------------------|--------------------|
| **Ввод темы** | Пользователь задаёт тему или промпт для презентации | Текстовое поле + настройки стиля (тон, длина, язык) + загрузка документов как контекст | Streamlit text_input + параметры (количество слайдов, стиль) |
| **Структурирование** | AI создаёт outline -- план слайдов с заголовками | GPT-4 генерирует иерархический план, пользователь может редактировать до генерации | LiteLLM вызывает LLM (Claude/GPT/Gemini) --> JSON outline с заголовками и ключевыми точками |
| **Генерация контента** | AI пишет текст для каждого слайда | GPT-4 + RAG (если загружены документы). Генерация по слайдам с учётом контекста всей презентации | LiteLLM --> markdown-контент для каждого слайда. Поддержка bullet points, нумерованных списков |
| **Дизайн** | Применение визуального шаблона и стилей | Собственный движок рендеринга: React-компоненты, темы, анимации, встроенные изображения | python-pptx + набор PPTX-шаблонов. Применение цветовых схем, шрифтов, макетов |
| **Экспорт** | Выгрузка готовой презентации | PDF, PPTX, HTML (link sharing с встроенным viewer, аналитика просмотров) | PPTX-файл для скачивания через Streamlit download_button |

## 5. Логика

Процесс клонирования шаг за шагом:

1. **Discovery** -- через скилл `/ai-search-free` нашли 20+ open-source аналогов Gamma. Поисковые запросы: "AI presentation generator open source", "Gamma.app alternative GitHub", "python-pptx AI slides". Результат: структурированный список из 20+ решений с метаданными.

2. **Оценка** -- составили таблицу сравнения по 6 критериям: язык (знакомый стек), лицензия (MIT/Apache), простота развёртывания (pip install vs Docker), AI-интеграция (какие LLM поддерживаются), активность разработки (последний коммит), качество документации.

3. **Выбор** -- SlideDeck AI победил по совокупности:
   - Python -- знакомый стек, можно быстро разобраться и модифицировать
   - MIT -- свободная лицензия без ограничений
   - LiteLLM -- поддержка 100+ LLM из коробки (Claude, GPT, Gemini, Mistral)
   - Streamlit -- простой UI без фронтенд-разработки
   - python-pptx -- нативная генерация PPTX без внешних зависимостей

4. **Аудит** -- агент `code-reviewer` проверил код: нет захардкоженных API-ключей (все через .env), нет SQL-инъекций (нет БД), зависимости актуальны, структура кода читаемая. Замечания: отсутствуют тесты, нет rate limiting на LLM-вызовы.

5. **Форк** -- минимальные изменения в конфигурации: настройка LiteLLM для работы через Claude API вместо дефолтного OpenAI. Остальной код без изменений -- он уже рабочий.

**Самое сложное:** выбор из 20+ решений. Presenton выглядит привлекательнее (4.8K stars, красивый UI), но его стек (Next.js + FastAPI + Docker + PostgreSQL) избыточен для учебного клона. SlideDeck AI -- это 80% функционала при 20% сложности.

## 6. Рвы

| Тип рва | Gamma.app | Клон (SlideDeck AI) |
|---------|-----------|---------------------|
| **Данные и шаблоны** | 1000+ профессиональных шаблонов, данные о предпочтениях миллионов пользователей, ML-модели обученные на реальном использовании | ~10 базовых PPTX-шаблонов из репозитория, нет данных о пользователях |
| **Бренд и доверие** | 5M+ пользователей, $20M+ инвестиций, упоминания в TechCrunch, Product Hunt #1, узнаваемость в нише | 346 stars на GitHub, известность только в open-source сообществе |
| **Switching cost** | Библиотека сохранённых презентаций, история генераций, командная работа (shared workspaces), интеграции (Notion, Slack) | Нулевой switching cost -- каждая генерация независима, нет аккаунта, нет хранилища |
| **Сетевой эффект** | Шеринг презентаций через ссылку (встроенный viewer), коллаборация в реальном времени, аналитика просмотров | Нет -- single-user инструмент, результат скачивается как файл |
| **Технология** | Проприетарный движок рендеринга (React-based), кастомные AI-пайплайны, CDN для быстрой загрузки | python-pptx (открытая библиотека), LiteLLM (открытый прокси), Streamlit (открытый фреймворк) |

**Вывод:** Главный ров Gamma -- комбинация данных, бренда и switching cost. 1000+ шаблонов невозможно воспроизвести без команды дизайнеров. Библиотека презентаций привязывает пользователя к платформе. Сетевой эффект (шеринг + коллаборация) создаёт дополнительный барьер.

Open-source решения вроде SlideDeck AI могут повторить базовый функционал (тема --> слайды --> PPTX), но не могут воспроизвести пользовательскую базу, данные для обучения моделей и экосистему интеграций. Это типичный паттерн: технология копируется за недели, а моат из данных и пользователей строится годами.
